# Desafios - Back-end

Desafios para as vagas de back-end da Studos.

- [Encurtador de URL](desafios/encurtador-de-url.md)

## Como executar os desafios

Para execução dos desafios, crie um repositório (público ou privado) e nos envie o endereço.
Caso prefira, você também pode nos enviar por email um arquivo compactado de seu projeto.
